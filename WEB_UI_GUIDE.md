## 📱 Web UI 功能详解

### 1️⃣ 💬 聊天界面（Chat）
最基础的聊天窗口

**功能：**
- 实时消息交互
- 自动添加到记忆库
- 支持 Shift+Enter 换行，Enter 发送

**使用示例：**
```
你: 你在干嘛？
AI女友: 我在想你呢~ 今天过得怎么样？

你: 挺好的，就是有点累
AI女友: 那就休息一下吧。我给你讲个笑话放松一下...
```

---

### 2️⃣ 📥 导入记录（Import）
一键导入微信或QQ的聊天历史

**支持的格式：**

#### JSON 格式（推荐）
```json
{
  "messages": [
    {
      "sender": "你的名字",
      "content": "你最近怎么样？",
      "timestamp": "2024-01-01 10:00:00"
    },
    {
      "sender": "AI女友",
      "content": "我很好，就是想你！",
      "timestamp": "2024-01-01 10:00:30"
    }
  ]
}
```

**步骤：**
1. 点击 "微信记录" 或 "QQ 记录" 按钮
2. 选择你的 JSON 聊天记录文件
3. 等待导入完成（会自动触发记忆提取）
4. 查看统计面板确认导入数据

**导入后的效果：**
- ✓ 数据库中增加对话记录
- ✓ 系统自动学习你的聊天风格
- ✓ 生成个性化的长期记忆
- ✓ AI 会记住你的偏好和习惯

---

### 3️⃣ 🧠 记忆库（Memory）
查看和管理 AI 学到的关于你的信息

**4个记忆分类：**

| 分类 | 内容示例 | 用途 |
|------|--------|------|
| **个性特征** (Personality) | "用户性格开朗，喜欢讲笑话" | AI 制造出适合的话题 |
| **关系记忆** (Relationship) | "用户很珍视我们的友谊" | 增加亲密感 |
| **经历记录** (Experience) | "用户上周去了旅游" | 后续对话中能引用 |
| **偏好习惯** (Preference) | "用户喜欢看电影，讨厌加班" | 个性化建议 |

**使用方式：**
1. 点击 "🔄 刷新" 加载最新记忆
2. 设置搜索关键词查找特定记忆
3. 每 50 条对话会自动提取长期记忆

---

### 4️⃣ 👩 个性化（Personality）
设置 AI 女友的性格和背景

**自定义项：**
```
AI名字: 萌萌
年龄: 24岁
职业: 自由插画师
城市: 北京
个性描述: 温柔体贴、略带调皮、爱撒娇但不做作

系统提示词: (高级)
你是一个温柔的女孩...
```

**提示：**
- 修改后点击 "💾 保存设置"
- 个性设置会影响 AI 的回复风格
- 系统提示词是最高优先级的指令

---

### 5️⃣ 📊 统计（Statistics）
查看聊天统计数据

**显示指标：**
- 📈 总对话数：累计聊天轮数
- 💬 用户消息数：你发送的消息总数
- 🤖 AI消息数：AI生成的回复总数
- 📏 平均回复长度：AI回复的平均字数

**用途：**
- 了解聊天频率
- 评估系统的有效性
- 优化设置参数

---

### 6️⃣ ⚙️ 设置（Settings）
系统参数调整和数据管理

**可调节参数：**

#### 回复温度 (0.0 - 1.0)
- **0.0** = 稳定保守（重复概率高）
- **0.5** = 均衡（推荐值）
- **1.0** = 创意十足（可能跑题）

#### 最大回复长度
- 50-2000 字符
- 更长 = 更详细但更慢

#### 使用内存增强
- ✓ 启用：利用你的历史记忆（推荐）
- ☐ 禁用：每次聊天都是"新认识"

**数据操作：**
- 🗑️ **清空所有数据** - 谨慎操作！
- 📥 **导出数据库** - 备份所有对话和记忆
- 📤 **导入数据库** - 恢复之前的备份

## 🔧 故障排除

### ❌ "连接错误"
**原因**: API 服务未启动  
**解决**:
```powershell
python run_server.py
# 看到 "Uvicorn running on http://0.0.0.0:8000" 说明成功
```

### ❌ "生成回复失败"
**原因**: Ollama 连接问题  
**解决**:
```powershell
# 诊断
python diagnose_ollama.py

# 重启 Ollama
ollama serve
```

### ❌ 导入记录失败
**原因**: JSON 格式不正确  
**解决**:
- 检查 JSON 是否完整（有效的 JSON 格式）
- 必须有 `"messages"` 数组
- 每条消息需要 `"sender"` 和 `"content"`

### ❌ 记忆库为空
**原因**: 还没有聊天或提取记忆  
**解决**:
- 先进行至少 5-10 次聊天
- 每 50 条对话会自动提取一次记忆
- 或手动导入历史记录

### 直接使用 API 而不是 Web UI

#### 聊天
```bash
curl -X POST "http://localhost:8000/chat" \
  -H "Content-Type: application/json" \
  -d '{

    "use_memory": true,
    "temperature": 0.7,
    "max_tokens": 512
  }'
```

```bash
curl "http://localhost:8000/memories"
```

#### 导入微信记录
```bash
curl -X POST "http://localhost:8000/import-chat/weixin" \
  -H "Content-Type: application/json" \
```

```bash
curl "http://localhost:8000/stats"
```
