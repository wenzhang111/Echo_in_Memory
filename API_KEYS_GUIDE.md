# API 密钥获取指南

本项目支持多个大模型 API，用户可以自由选择使用。所有 API 密钥都存储在浏览器本地，不会上传到服务器。

## 🚀 推荐方案：DeepSeek（最快最便宜）

### DeepSeek API
**优势**：
- 速度极快（1-3秒回复）
- 价格最便宜（约 OpenAI 的 1/10）
- 中文支持优秀
- 支持长上下文

**获取步骤**：
1. 访问 [DeepSeek 开放平台](https://platform.deepseek.com)
2. 注册账号（可用邮箱）
3. 在"API Keys"页面创建新的 API Key
4. 复制密钥，粘贴到 Web UI 设置页面

**初始额度**：新用户通常获得 $5 免费额度

---

## ⚡ 备选方案：OpenAI（功能最强）

### GPT-3.5-turbo（推荐）
**优势**：
- 回复质量好
- 速度快（2-5秒）
- 价格中等

**获取步骤**：
1. 访问 [OpenAI 平台](https://platform.openai.com)
2. 注册账号或登录
3. 进入 "API Keys" 页面
4. 点击 "Create new secret key"
5. 复制密钥到 Web UI

**计费**：按使用量计费，需要绑定支付方式（信用卡/Visa）

---

### GPT-4-turbo（更强大）
**优势**：
- 回复质量最好
- 理解能力更强
- 支持更长的上下文

**价格**：比 GPT-3.5 贵 10 倍左右

## ✨ 高级方案：Claude（最聪慧）

### Claude 3 Sonnet
**优势**：
- 回复最自然流畅
- 理解最深入
- 适合长篇对话

**获取步骤**：
1. 访问 [Anthropic 控制台](https://console.anthropic.com)
2. 注册账号

**计费**：按 token 计费，初期有免费额度

## 🏠 本地方案：Ollama（完全免费）

**优势**：
- 完全免费，无限制使用
- 完全隐私，数据不上传
- 离线运行，无网络依赖

**劣势**：
- 速度取决于本地硬件
- RTX 4060 运行 Qwen3:4b 需 10-60 秒

**使用**：在 Web UI 设置页面选择"本地(Ollama)"模型即可

## 🔐 安全提示

- ✅ 密钥只存在浏览器 localStorage 中
- ✅ 不会发送给第三方服务器
- ✅ 每次请求直接调用 API
- ⚠️ 不要将密钥分享给他人
- ⚠️ 如果泄露，立即在对应平台更新密钥

---

## 🆘 常见问题

**Q: 需要配置所有 API 吗？**
A: 不需要，选一个就够了。页面会自动检测可用的模型。

**Q: 密钥过期了怎么办？**
A: 在对应平台重新生成，粘贴到 Web UI 设置页面即可。

**Q: 可以同时用多个 API 吗？**
A: 可以，在设置页面配置多个，然后在聊天时选择想要使用的模型。

**Q: 为什么有些模型显示"不可用"？**
A: 因为没有配置对应的 API 密钥，或密钥无效。

## 📞 获取帮助

如果配置有问题，请：
1. 检查 API 密钥是否正确复制（尤其是前缀 `sk-`）
2. 确保密钥不包含多余空格
3. 在 Web UI 点击"检测可用模型"按钮诊断
4. 查看浏览器控制台（F12）的错误信息
