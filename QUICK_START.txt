⚠️  常见问题快速解决
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

问题: Web UI 无法连接到 API
原因: 服务器未启动
解决: python run_server.py (看到 "Uvicorn running" 即可)

问题: "生成回复失败"
原因: Ollama 未运行或模型未加载
解决: 
  1. ollama serve (启动 Ollama)
  2. ollama list (检查模型)
  3. python diagnose_ollama.py (诊断)

问题: "导入失败 - 格式错误"
原因: JSON 格式不正确
解决: 
  确保有 "messages" 键
  每条消息有 "sender" 和 "content"
  参考 WEB_UI_GUIDE.md 的 JSON 示例

问题: 记忆库是空的
原因: 需要聊天或导入历史
解决:
  先进行 10+ 次对话，或
  导入你的聊天历史

🎓 使用流程示例
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【新用户上手】
1. 启动系统 (start_complete.bat)
2. 打开 web_ui.html
3. 去 "👩 个性化" 设置 AI 名字
4. 开始在 "💬 聊天" 里聊天
5. 每 50 条对话会自动提取记忆

【导入历史】
1. 准备 JSON 格式的聊天记录
2. 打开 web_ui.html
3. 去 "📥 导入记录" 上传文件
4. 等待导入完成（自动触发记忆提取）
5. 去 "📊 统计" 验证导入成功

【优化效果】
1. 去 "⚙️  设置" 调整参数
   - 温度: 0.3-0.4 稳定, 0.7 均衡, 0.9+ 创意
   - 长度: 300-500 简短, 800+ 详细
2. 去 "🧠 记忆库" 查看学到的信息
3. 去 "📊 统计" 监测对话量
4. 定期导出数据备份
